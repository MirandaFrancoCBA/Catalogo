# 🌿 Catálogo El Laberinto

Catálogo web de productos artesanales en **arcilla, yeso y cemento**.
Diseñado para mostrar productos de forma simple, visual y profesional, con opción de contacto directo por WhatsApp.

---

## 🚀 Demo

👉 (Agregá acá tu link de GitHub Pages)
Ejemplo:
https://tuusuario.github.io/catalogo/

---

## 📸 Características

* 🔎 Búsqueda en tiempo real
* 🎯 Filtros por categoría, tipo, estado y precio
* 📊 Ordenamiento de productos
* 🖼️ Modal con galería de imágenes
* 🌙 Modo oscuro con persistencia
* 🌐 Soporte multi-idioma (ES / EN)
* 📱 Diseño responsive (mobile-first)
* 💬 Botón de contacto por WhatsApp
* ⚡ Carga de datos desde JSON (fácil de mantener)

---

## 🧱 Tecnologías utilizadas

* HTML5
* CSS3 (Flexbox + Grid)
* JavaScript (Vanilla JS)
* JSON como base de datos

## 🛠️ Personalización

Podés editar fácilmente el catálogo desde:

### 📦 `data/productos.json`

Ejemplo de producto:

```json
{
  "id": 1,
  "nombre": "Maceta pequeña clásica",
  "categoria": "cerámica",
  "tipo": "maceta",
  "estado": "crudo",
  "precio": 2500,
  "imagenes": [
    "ceramica/maceta_pequena.jpg"
  ],
  "descripcion": "Ideal para cactus o suculentas."
}
```

---

## 📷 Recomendaciones de imágenes

* Usar fondo neutro o natural
* Mantener mismo estilo en todas las fotos
* Resolución optimizada (no muy pesadas)
* Nombres de archivos sin espacios ni mayúsculas

Ejemplo:

```
maceta_pequena_1.jpg
```
